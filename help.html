How to use ProfitPilot

ProfitPilot is a decision tool: it helps you estimate profit/margin, reverse-calculate a required price, and run bulk checks quickly — based on your own inputs.
Open app Log in / Sign up
Step 1
Choose currency + optional FX
In the app, choose a display currency. If you sell in one currency but buy in another, enter an exchange rate manually (no live rates).
Tip: If you don’t need conversion, leave FX empty.
Step 2
Pick a preset or manual
Presets are examples. You can always override values manually. The output is only as accurate as the fee/return assumptions you enter.
Important: Presets are not “official fees”. They’re reference examples.
Mode
Profit mode (default)
Fill in:
  • Selling price
  • Product cost + shipping + packaging/misc
  • Fee %, fixed fee, ads %, return rate %
Then press Calculate.
Read the KPIs: Profit + margin are your key decision numbers. Fees/ads shows the bleed.
Mode
Reverse mode (target profit → required price)
Use this when you know the profit you want per order. Enter a Target profit and ProfitPilot estimates the selling price required to hit it.
Trial limitation: Reverse mode is limited during trial (so users taste it). PRO unlock gives unlimited reverse calculations.
Mode
Bulk mode (table)
Bulk mode is for quickly checking many product ideas. You can:
  • Add rows and type values into the table
  • Paste rows (Sell, Cost, Ship, Misc) then import
  • Apply current fees to all rows
  • Run bulk to compute profit/margin per row
  • Export bulk CSV
Best practice: Use Bulk to shortlist winners, then open Profit mode for deeper analysis.
Feature
Insights (PRO)
Insights are decision signals to reduce bad listings:
  • Break-even price
  • Price to hit target margin
  • Max ads % you can afford before loss
  • Risk hints based on margin sensitivity
Note: These are not guarantees — they’re estimates from your inputs.
Feature
Export & sharing
Export helps you keep records or share a calculation:
  • CSV export (useful for analysis)
  • PDF export (print-friendly; PRO)
  • Share link (PRO)
Reviews
Leave feedback (public)
Reviews are visible publicly on Reviews. To submit or edit your review, log in and use the review box inside the app.
